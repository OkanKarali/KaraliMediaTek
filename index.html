Bağlantı Hatası

Uygulama, veritabanına bağlanamadı. Bu genellikle API anahtarı kısıtlamalarından kaynaklanır.

Çözüm Adımları:

  1. Google Cloud Console'da projenizin "Kimlik Bilgileri" sayfasına gidin.
  2. Kullandığınız API Anahtarını seçin.
  3. "Uygulama kısıtlamaları" bölümünde "Web siteleri" seçeneğinin işaretli olduğundan emin olun.
  4. "Web sitesi kısıtlamaları" altında, okankarali.github.io adresinin izin verilenler listesinde olduğundan emin olun. Eğer değilse ekleyin.
